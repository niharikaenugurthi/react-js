
# React Gallery App

A simple and responsive gallery application built using React, Axios, and Tailwind CSS.

The app fetches images dynamically from the Picsum API and displays them in a scrollable gallery with pagination support.

---

## 🚀 Features

- Fetch images using API
- Pagination (Next & Prev buttons)
- Responsive gallery layout
- Scrollable image section
- Loading state
- Tailwind CSS styling
- React Hooks usage
- Axios for API requests

---

## 🛠 Technologies Used

- React JS
- Axios
- Tailwind CSS
- Picsum API

---
