# React Notes App

A simple and responsive Notes Application built using React.

This app allows users to create and delete notes dynamically with a clean user interface.

---

## 🚀 Features

- Add Notes
- Delete Notes
- Dynamic Note Rendering
- Responsive UI
- React Hooks usage
- Simple and clean design

---

## 🛠 Technologies Used

- React JS
- JavaScript
- Tailwind CSS
