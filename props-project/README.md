# React Job Listing App

A modern and responsive Job Listing Application built using React and Tailwind CSS.

This application displays multiple job cards dynamically using reusable components and props.

---

## 🚀 Features

- Dynamic Job Listings
- Reusable Card Components
- Responsive UI Design
- Company Logos
- Job Tags
- Salary Information
- Location Details
- Clean Component Structure

---

## 🛠 Technologies Used

- React JS
- JavaScript
- Tailwind CSS
