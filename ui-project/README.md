# React Profile Card UI

A modern and responsive Profile Card UI built using React and Tailwind CSS.

This project dynamically displays user cards with images, colored tags, and profile information using reusable components and props.

---

## 🚀 Features

- Dynamic User Cards
- Responsive UI Design
- Tailwind CSS Styling
- Hover Animations
- Dynamic Tag Colors
- Reusable Components
- Props-Based Data Rendering

---

## 🛠 Technologies Used

- React JS
- JavaScript
- Tailwind CSS
