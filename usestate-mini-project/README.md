# React Counter App

A simple Counter Application built using React and useState Hook.

This project allows users to increase, decrease, and reset the counter dynamically.

---

## 🚀 Features

- Increase Counter
- Decrease Counter
- Reset Counter
- Dynamic UI Updates
- React Hooks Usage
- Simple and Clean Design

---

## 🛠 Technologies Used

- React JS
- JavaScript
- CSS
